Work Process

기본 작업 기간 ~5일

기본 수정 2회 (작은 범위 수정 무료)

빠른/당일마감 가능

  1. 문의

    양식 작성 및 견적가 안내

  2. 입금

    입금 확인 후 작업 시작

  3. 1차 시안(컨셉)

    레이아웃, 컨셉 디자인 시안

  4. 컨펌/수정

    피드백 반영 후 CSS 이식

  5. 2차 시안(CSS)

    애니메이션과 실제 적용 화면 확인

  6. 컨펌/수정

    2차 수정사항 반영

  7. 전달

    파일·적용 가이드 전달

작업 전 유의사항

  • ✦ OBS 및 위플랩 세팅이 어려우시다면 원격을 통한 설치 대행은 무료로 진행합니다.

  • ✦ 다양한 브라우저에 대응 가능합니다. 별도 언급이 없으시면 위플랩 기준으로 작업합니다.

  • ✦ 양도/재판매/소스 코드 배포를 엄격히 금지합니다.

  • ✦ 모든 작업물은 포트폴리오/샘플로 활용될 수 있습니다.

  • ✦ 디자인을 별도로 받아오시는 경우 디자인 작가님께 양해를 구한 후 작업이 진행됩니다.

  • ✦ 채팅 화면 자체는 제작하지 않습니다. 사용 중 화면을 첨부해 주세요

  • ✦ 오마카세 디자인의 경우 큰 폭의 수정은 거절될 수 있습니다.
